# Sprint-2-Prompt & AI

## Objetivo
Desenvolver um chatbot funcional utilizando IA para auxiliar usuários do sistema EV ChargeOps da GoodWe.

## Tecnologias
- Python 3
- Google Gemini API
- Google Colab

## Instalação
pip install google-generativeai

## Execução
1. Configure sua chave da API Gemini.
2. Execute o arquivo chatbot.py.
3. Faça perguntas relacionadas ao EV ChargeOps.
4. Digite 'sair' para encerrar.

## Casos de Teste
1. Como funciona o compartilhamento de carregadores?
2. Quem paga pela energia utilizada?
3. O sistema registra as recargas?
4. Posso consultar meu histórico de consumo?
5. Qual a principal vantagem do EV ChargeOps?
