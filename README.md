# Sobre o Projeto

O GoodWe EV Assistant é um chatbot desenvolvido para auxiliar usuários da solução EV ChargeOps da GoodWe.

O sistema foi criado para responder dúvidas relacionadas ao gerenciamento de carregadores de veículos elétricos em condomínios, auxiliando síndicos, moradores e operadores.

O chatbot utiliza uma abordagem baseada em regras e palavras-chave para fornecer respostas rápidas e contextualizadas sobre o funcionamento da plataforma.

---

# Problema Abordado

Com o crescimento do uso de veículos elétricos, condomínios e empresas enfrentam desafios relacionados ao gerenciamento dos carregadores, controle de consumo de energia e faturamento dos usuários.

A solução EV ChargeOps busca facilitar esse gerenciamento, permitindo o compartilhamento de carregadores e o acompanhamento das recargas realizadas.

---

# Objetivo

- Responder dúvidas sobre carregadores elétricos;
- Explicar o compartilhamento de carregadores;
- Informar sobre consumo de energia;
- Explicar faturamento;
- Fornecer histórico de recargas;
- Atender usuários de forma simples e objetiva.

# Como Executar

1. Instale o Python 3.
2. Abra o terminal na pasta do projeto.
3. Execute:

python chatbot.py

4. Digite suas perguntas.
5. Digite 'sair' para encerrar.
